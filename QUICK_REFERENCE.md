## 表4: 参数与诊断的关系

### 4.1 欠/过阻尼判断

**临界条件**: $c^2 = 4km$

当前模型: $c^2 = 3.5^2 = 12.25 < 4 \times 12 \times 1 = 48$

**结论**: **欠阻尼** (正常状态) ✓

| 诊断 | c² vs 4km | 特征 | 对应症状 |
|------|----------|------|----------|
| 欠阻尼 | c² < 4km | 易振荡 | 焦虑、惊恐、易激动 |
| **临界阻尼** | **c² ≈ 4km** | **最优** | **健康稳定** |
| 过阻尼 | c² > 4km | 反应迟缓 | 抑郁、无动力、反刍 |

```

c_new = c₀ × α_c
```

## 表8: 事件影响速查

### 8.1 应激事件

**基础冲击**: -30.0  
**调节因素**:
- 事件严重程度 (0-10) ÷ 5
- 睡眠压力调制: 1.0 + 0.5×S
- 神经稳定性调制: 1.0 + (5-c)/5

**示例**:
```
中等应激(severity=5), S=0.5, c=3.5:
impulse = -30 × (5/5) × (1+0.5×0.5) × (1+(5-3.5)/5)
        = -30 × 1.0 × 1.25 × 1.3
        = -48.75
        
高应激(severity=8), S=0.8, c=2.0:
impulse = -30 × (8/5) × (1+0.5×0.8) × (1+(5-2.0)/5)
        = -30 × 1.6 × 1.4 × 1.6
        = -107.5  (严重冲击!)
```

### 8.2 运动事件

**基础冲击**: +25.0  
**强度响应**: (intensity/5.0)^0.7

**示例**:
```
中等强度运动(intensity=5):
impulse = +25 × (5/5)^0.7 = +25 × 1.0 = +25.0

高强度运动(intensity=8):
impulse = +25 × (8/5)^0.7 = +25 × 1.32 = +33.0

低强度运动(intensity=3):
impulse = +25 × (3/5)^0.7 = +25 × 0.73 = +18.25
```

## 表10: 参考值速查

### 10.1 临界值

```
阻尼临界值: c_crit² = 4km = 4×12×1 = 48
当前: c² = 12.25 < 48 ✓ 欠阻尼

睡眠压力临界值:
- S > 0.8: 严重睡眠不足
- S > 0.5: 中等睡眠不足
- S < 0.2: 充分休息

情绪位移临界值:
- x < -0.8: 严重反刍
- x < -0.5: 中度反刍
- -0.2 < x < 0.2: 正常
- x > 0.5: 积极状态
```

### 10.2 时间尺度

```
情绪恢复时间 (1/2 衰减): 
t_{1/2} = -m/c × ln(0.5) ≈ 0.2 小时 = 12分钟

睡眠压力恢复 (90% 清除):
8小时睡眠清除约78%
需要8.5小时清除90%

昼夜节律适应:
- 相邻时区(3h差): 3-5天
- 远程时差(8h): 7-10天  
- 极端时差(12h): 21天+
```
